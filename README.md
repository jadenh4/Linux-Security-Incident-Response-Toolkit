# Linux-Security-Incident-Response-Toolkit

This project contains Bash scripts designed to assist with basic system monitoring, file auditing, and incident response tasks on Linux/macOS systems.

# Key Features
- Process enumeration and termination
- World-writable file detection
- Log archival with timestamping
- Suspicious process and open port scanning

# Technologies
- Bash
- Linux CLI tools (ps, find, lsof, tar)
